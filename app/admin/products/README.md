# 상품관리 페이지 사용법

## 🎯 주요 기능

### 1. 상품 추가
- **새 상품 추가** 버튼을 클릭하여 새로운 상품을 등록할 수 있습니다
- 상품명, 가격, 카테고리, 설명, 이미지 등을 입력할 수 있습니다
- 이미지는 드래그 앤 드롭으로 업로드하거나 URL을 직접 입력할 수 있습니다
- **즉시 웹 적용**: 상품 추가 후 홈페이지에 즉시 반영됩니다

### 2. 상품 편집
- 각 상품의 **편집** 버튼(연필 아이콘)을 클릭하여 상품 정보를 수정할 수 있습니다
- 모든 필드를 수정할 수 있으며, 변경사항은 즉시 저장됩니다
- **즉시 웹 적용**: 수정된 내용이 홈페이지에 즉시 반영됩니다

### 3. 상품 삭제
- 개별 삭제: 각 상품의 **삭제** 버튼(휴지통 아이콘)을 클릭
- 일괄 삭제: 여러 상품을 체크박스로 선택 후 **일괄 삭제** 버튼 클릭
- **즉시 웹 적용**: 삭제된 상품이 홈페이지에서 즉시 제거됩니다

### 4. 상품 검색 및 필터링
- **검색**: 상품명이나 설명으로 검색 가능
- **카테고리 필터**: 특정 카테고리만 표시
- **정렬**: 이름순, 가격순, 카테고리순으로 정렬 가능
- **정렬 순서**: 오름차순/내림차순 선택 가능

### 5. 일괄 작업
- **일괄 선택**: 헤더의 체크박스로 모든 상품 선택/해제
- **개별 선택**: 각 상품의 체크박스로 개별 선택
- **일괄 재고 변경**: 선택된 상품들을 한 번에 재고 있음/품절로 변경
- **일괄 삭제**: 선택된 상품들을 한 번에 삭제

### 6. 재고 상태 관리
- **빠른 변경**: 재고 상태 버튼을 클릭하여 즉시 상태 변경
- **시각적 표시**: 재고 있음(초록색), 품절(빨간색)으로 구분
- **즉시 웹 적용**: 재고 상태 변경이 홈페이지에 즉시 반영됩니다

## 🚀 **웹 적용 방법**

### **즉시 반영 시스템**
상품을 추가, 수정, 삭제하면 **즉시** 홈페이지와 상품 페이지에 반영됩니다:

1. **상품 추가** → 홈페이지 상품 목록에 즉시 표시
2. **상품 수정** → 가격, 설명, 이미지 등 즉시 업데이트  
3. **상품 삭제** → 웹에서 즉시 제거
4. **재고 상태 변경** → 즉시 반영

### **데이터 저장 방식**
- **Zustand 스토어**: React 상태 관리로 즉시 반영
- **로컬스토리지**: 브라우저에 자동 저장되어 새로고침 시에도 유지
- **실시간 동기화**: 모든 페이지에서 동일한 상품 데이터 사용

### **별도 작업 불필요**
- ❌ 서버 재시작 불필요
- ❌ 배포 과정 불필요  
- ❌ 데이터베이스 동기화 불필요
- ❌ 캐시 정리 불필요

## 🖼️ 이미지 업로드

### 지원 형식
- PNG, JPG, GIF
- 최대 파일 크기: 5MB

### 업로드 방법
1. **드래그 앤 드롭**: 이미지 파일을 업로드 영역에 드래그
2. **파일 선택**: "이미지 선택" 버튼 클릭하여 파일 선택
3. **URL 입력**: 이미지 URL을 직접 입력

### **바로 업로드 기능**
- 이미지 업로드 후 **즉시 미리보기** 표시
- 상품 저장 시 **자동으로 홈페이지에 반영**
- **실시간 이미지 변경** 지원

## 📊 상품 통계

페이지 하단에 다음 통계 정보가 표시됩니다:
- 총 상품 수
- 재고 있는 상품 수
- 품절 상품 수
- 평균 가격

## 🔍 **홈페이지 확인 방법**

### **1. 헤더 버튼**
- **"홈페이지에서 확인"** 버튼을 클릭하여 새 탭에서 홈페이지 열기

### **2. 개별 상품 확인**
- 각 상품의 **눈 아이콘**을 클릭하여 홈페이지에서 해당 상품 확인

### **3. 실시간 모니터링**
- 상품 변경 후 **즉시** 홈페이지에서 결과 확인 가능
- **새로고침 없이** 변경사항 확인 가능

## ⚠️ 주의사항

1. **삭제 확인**: 상품 삭제 시 확인 메시지가 표시됩니다
2. **데이터 저장**: 모든 변경사항은 자동으로 저장됩니다
3. **이미지 URL**: 외부 이미지 URL이 유효한지 확인하세요
4. **가격 입력**: 가격은 소수점 둘째 자리까지 입력 가능합니다
5. **즉시 반영**: 변경사항이 즉시 웹에 반영되므로 신중하게 작업하세요

## 🔧 기술적 특징

- **반응형 디자인**: 모바일과 데스크톱에서 모두 사용 가능
- **실시간 검색**: 타이핑과 동시에 검색 결과 표시
- **드래그 앤 드롭**: 직관적인 이미지 업로드
- **일괄 작업**: 효율적인 상품 관리
- **상태 관리**: Zustand를 통한 전역 상태 관리
- **즉시 반영**: 모든 변경사항이 웹에 즉시 적용
- **자동 저장**: 로컬스토리지에 자동 동기화

## 🌐 **웹 적용 확인 체크리스트**

상품 변경 후 다음을 확인하세요:

- [ ] **홈페이지**에서 상품이 올바르게 표시되는지 확인
- [ ] **상품 페이지**에서 상품 정보가 정확한지 확인  
- [ ] **이미지**가 올바르게 로드되는지 확인
- [ ] **가격과 설명**이 정확한지 확인
- [ ] **재고 상태**가 올바르게 표시되는지 확인
- [ ] **새로고침** 후에도 데이터가 유지되는지 확인
