# 히어로 슬라이드 이미지 선명도 및 관리자 수정 가이드

## 1. 적용된 변경 사항 (모든 슬라이드에 공통)

- **배경 이미지 선명도**: 카테고리 히어로(Stickers, Stamps, Phone Cases, Hot Goods)와 메인 홈페이지 히어로 슬라이드 모두에 아래가 반영되었습니다.
  - **기본 투명도(Opacity)**: 슬라이드별 기본값을 **100%(1)** 로 변경해 이미지가 어둡게 보이지 않도록 했습니다.
  - **오버레이 완화**: 이미지를 가리던 진한 그라데이션 오버레이를 제거하고, **텍스트 가독용으로만** 하단에 약한 그라데이션(`from-black/20`)만 적용했습니다.
  - **카테고리 슬라이드**: `SlidingBackground` 컴포넌트에서 2x 렌더 후 축소 방식으로 고해상도 디스플레이에서 더 선명하게 보이도록 했습니다.

---

## 2. 관리자가 슬라이드를 수정하는 방법

### 2-1. 카테고리 히어로 슬라이드 (Stickers / Stamps / Phone Cases / Hot Goods)

1. **관리자 로그인** 후 **Admin** 메뉴로 이동합니다.
2. **Content** (콘텐츠 관리) 메뉴를 클릭합니다.
3. 왼쪽에서 해당 카테고리를 선택합니다.
   - 예: **Stickers** → **Stickers Hero Background Slides**
   - 또는 **Stamps**, **Phone Cases**, **Hot Goods** 각각의 **Hero Background Slides** 섹션
4. 슬라이드 목록에서 **Edit** (연필 아이콘)를 눌러 수정하거나, **Add Slide**로 새 슬라이드를 추가합니다.

**Edit Slide / Add Slide 화면에서 조정 가능한 항목:**

| 항목 | 설명 | 선명도 관련 |
|------|------|-------------|
| **Image / Video** | 배경 이미지 또는 동영상 URL/업로드 | 고해상도 이미지를 사용할수록 선명합니다. |
| **Slide Title (Optional)** | 슬라이드 제목 | 비우면 제목 없이 이미지만 표시됩니다. |
| **Slide Subtitle (Optional)** | 슬라이드 부제목 | 비우면 부제목 없이 표시됩니다. |
| **Opacity** | 슬라이드 이미지/동영상의 밝기 (0~100%) | **기본값 100%** 로 설정되어 있습니다. 이미지를 선명하게 보이게 하려면 **100%** 를 유지하고, 텍스트 가독을 위해 살짝 어둡게 하려면 80~90% 정도로 낮출 수 있습니다. |
| **Speed** | 슬라이드 전환/자동 재생 속도 (1–10) | 선명도와 무관. |
| **Effect** | 전환 효과 (slide / fade / zoom 등) | 선명도와 무관. |
| **Direction** | 슬라이드 이동 방향 (left / right / up / down) | Effect가 slide일 때만 적용. |
| **Responsive (Mobile / Tablet / Desktop)** | 기기별 **Speed**, **Opacity** | 모바일/태블릿/데스크톱마다 **Opacity**를 다르게 줄 수 있습니다. 기본값은 **100%** 입니다. |

- **선명도를 유지하려면**: Opacity를 **100%** 에 두고, 제목/부제목이 잘 안 보일 때만 80~95% 정도로 낮추는 것을 권장합니다.
- **새로 추가하는 슬라이드**는 기본 Opacity가 **100%** 로 설정됩니다.

---

### 2-2. 메인 홈페이지 히어로 슬라이드

1. **Admin** → **Content** 이동.
2. **Main Hero Slides** (메인 히어로 슬라이드) 섹션을 선택합니다.
3. 각 슬라이드의 **제목, 링크, 색상** 등을 수정할 수 있습니다.
4. 메인 히어로는 **진한 오버레이가 제거**되어, 배경 이미지가 이전보다 선명하게 보이도록 되어 있습니다. 텍스트 가독을 위해 **하단만 살짝 어둡게** 하는 그라데이션만 적용되어 있습니다.

---

## 3. 요약

- **모든 슬라이드**(카테고리 + 메인)에 **선명도 개선**(기본 Opacity 100%, 오버레이 완화)이 반영되어 있습니다.
- **수정 위치**: Admin → Content → (카테고리 선택) → **Hero Background Slides** → **Edit** 또는 **Add Slide**.
- **Opacity** 로 이미지 밝기(선명도 체감)를 조절할 수 있으며, **100%** 가 기본값이며 선명도 유지에 유리합니다.
- 기기별로 다르게 하려면 **Responsive** 탭에서 Mobile/Tablet/Desktop 각각의 **Opacity**를 설정하면 됩니다.
