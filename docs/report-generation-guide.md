## 📄 보고서 생성 방법

자동화 생산 플랫폼 관련 문서를 보고서 형식으로 변환하는 여러 방법을 안내합니다.

---

## 방법 1: 마크다운을 PDF로 변환 (추천 ⭐)

### 장점
- ✅ 간단하고 빠름
- ✅ 전문적인 문서 형식
- ✅ 공유 및 인쇄 용이
- ✅ 기존 홈페이지에 영향 없음

### 도구 옵션

#### 옵션 A: VS Code 확장 프로그램 사용
1. **Markdown PDF** 확장 프로그램 설치
   - VS Code에서 `Ctrl + Shift + X` (확장 프로그램)
   - "Markdown PDF" 검색 및 설치
2. 사용 방법
   - 마크다운 파일 열기
   - `Ctrl + Shift + P` → "Markdown PDF: Export (pdf)" 선택
   - PDF 파일 자동 생성

#### 옵션 B: 온라인 도구 사용
- **Markdown to PDF**: https://www.markdowntopdf.com/
- **Dillinger**: https://dillinger.io/
- **StackEdit**: https://stackedit.io/

#### 옵션 C: 명령줄 도구 (고급)
```bash
# Pandoc 설치 필요
npm install -g md-to-pdf
md-to-pdf docs/automated-production-platform-design.md
```

## 방법 2: HTML 보고서 생성

### 장점
- ✅ 웹 브라우저에서 바로 확인 가능
- ✅ 인터랙티브한 링크 및 네비게이션
- ✅ 스타일링 커스터마이징 가능

### 구현 방법

#### 옵션 A: 간단한 HTML 변환
마크다운을 HTML로 변환하는 스크립트 생성

#### 옵션 B: 정적 사이트 생성기 사용
- **MkDocs**: Python 기반 문서 사이트 생성기
- **Docusaurus**: React 기반 문서 사이트
- **GitBook**: 전문적인 문서 플랫폼

---

## 방법 3: Word 문서로 변환

### 장점
- ✅ Microsoft Word에서 편집 가능
- ✅ 비즈니스 문서 표준 형식
- ✅ 공유 및 협업 용이

### 도구
- **Pandoc**: 마크다운을 Word로 변환
  ```bash
  pandoc input.md -o output.docx
  ```
- **온라인 변환기**: https://cloudconvert.com/md-to-docx
