# SELPIC-X 문서 안내

## ⚠️ 중요 안내

이 폴더의 SELPIC-X 관련 문서들은 **구상 단계의 학습 자료**입니다.

### 현재 상태
- ✅ **구상 및 설계 완료**: 비즈니스 모델, 아키텍처, 데이터 모델 등 설계 완료
- ⏸️ **구현 보류**: 아직 실제 개발은 진행하지 않음
- 🔒 **기존 홈페이지와 분리**: 현재 홈페이지 개발에 전혀 영향 없음

### 문서 목록

1. **automated-production-platform-planning.md**
   - 구상 단계 개요 및 학습 가이드
   - 현재 상태 및 향후 계획

2. **automated-production-platform-design.md**
   - 전체 시스템 설계 문서
   - 비즈니스 모델, 아키텍처, 데이터 모델, API 설계 등

3. **automated-production-platform-implementation-guide.md**
   - 구현 가이드 (참고용)
   - 데이터베이스 스키마, 코드 예제 등

### 사용 방법

1. **학습 목적**: 설계 내용을 학습하고 검토
2. **구상 검토**: 비즈니스 모델 및 기술적 접근 방식 검토
3. **향후 참고**: 구현 단계에서 참고 자료로 활용

### 주의사항

- ❌ **현재는 구현하지 않음**: 코드 작성 및 개발 보류
- ✅ **문서만 관리**: 설계 문서와 학습 자료만 유지
- 🔒 **기존 홈페이지 보호**: 현재 개발 중인 홈페이지에 영향 없음
- 🔗 **나중에 통합**: 안정화 후 단계적 통합 예정
