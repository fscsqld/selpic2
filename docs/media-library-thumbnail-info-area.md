# Media Library 썸네일 하단 어두운 영역 설명

## 📋 개요

Media Library 모달의 그리드 뷰에서 각 이미지/동영상 썸네일 하단에 어두운 그라디언트 영역이 표시됩니다. 이 영역은 **파일 정보를 표시하기 위한 UI 요소**입니다.

---

## 🎨 어두운 영역의 역할

### 1. **파일 정보 표시 영역**

```typescript
<div className="absolute bottom-0 left-0 right-0 bg-gradient-to-t from-black/95 via-black/80 to-transparent text-white p-2">
  <p className="text-xs font-medium truncate mb-0.5">{file.name}</p>
  <div className="flex items-center gap-1 text-[11px] text-gray-200">
    <Folder className="w-2.5 h-2.5" />
    <span className="truncate">{categories.find(c => c.id === file.category)?.name || file.category}</span>
  </div>
</div>
```

### 2. **표시되는 정보**

#### 파일명
- **위치**: 상단
- **스타일**: 흰색 텍스트, `text-xs`, `font-medium`
- **기능**: 파일 이름 전체 표시 (긴 이름은 `truncate`로 생략)

#### 카테고리
- **위치**: 하단
- **스타일**: 연한 회색 텍스트 (`text-gray-200`), `text-[11px]`
- **아이콘**: 폴더 아이콘 (`Folder`)
- **기능**: 파일이 속한 카테고리 표시
  - 예: "Stickers", "Stamps", "Phone Cases", "HOT GOODS", "General"

---

## 🎨 디자인 특징

### 1. **그라디언트 배경**

```css
bg-gradient-to-t from-black/95 via-black/80 to-transparent
```

#### 의미
- **`bg-gradient-to-t`**: 아래에서 위로 그라디언트
- **`from-black/95`**: 하단 95% 불투명도의 검은색
- **`via-black/80`**: 중간 80% 불투명도의 검은색
- **`to-transparent`**: 상단은 완전 투명

#### 효과
- 썸네일 이미지 위에 텍스트가 잘 보이도록 어두운 배경 제공
- 상단으로 갈수록 투명해져서 이미지와 자연스럽게 블렌딩
- 텍스트 가독성 향상

### 2. **레이아웃**

```typescript
className="absolute bottom-0 left-0 right-0"
```

- **절대 위치**: 썸네일 컨테이너 하단에 고정
- **전체 너비**: `left-0 right-0`로 썸네일 전체 너비 사용
- **패딩**: `p-2` (8px)로 내부 여백 제공

---

## 💡 왜 어두운 배경을 사용하나요?

### 1. **텍스트 가독성**
- 이미지 색상이 밝거나 복잡할 때 흰색 텍스트가 잘 보이지 않을 수 있음
- 어두운 배경으로 텍스트 대비를 확보

### 2. **정보 강조**
- 파일명과 카테고리 정보를 명확히 구분
- 사용자가 빠르게 파일 정보를 파악 가능

### 3. **시각적 일관성**
- 모든 썸네일에 동일한 스타일 적용
- 깔끔하고 전문적인 UI 제공

---

## 🔍 실제 사용 예시

### 표시 예시

```
┌─────────────────────┐
│                     │
│   [이미지 썸네일]    │
│                     │
│                     │
├─────────────────────┤ ← 어두운 그라디언트 영역 시작
│ product-image.jpg   │ ← 파일명 (흰색)
│ 📁 Stickers         │ ← 카테고리 (연한 회색)
└─────────────────────┘
```

### 그라디언트 효과

```
투명 (상단)
    ↓
약간 어두움 (중간)
    ↓
매우 어두움 (하단) ← 텍스트 영역
```

---

## 🎯 개선 가능한 부분

### 현재 구조
- 파일명과 카테고리만 표시
- 간단하고 깔끔한 디자인

### 추가 가능한 정보
- 파일 크기
- 업로드 날짜
- 파일 타입 아이콘 (이미지/동영상)
- 태그 (있는 경우)

---

## 📝 요약

### 어두운 영역의 목적
✅ **파일 정보 표시**: 파일명과 카테고리  
✅ **텍스트 가독성**: 어두운 배경으로 흰색 텍스트 가독성 향상  
✅ **시각적 일관성**: 모든 썸네일에 동일한 스타일 적용  

### 디자인 특징
- **그라디언트**: 검은색에서 투명으로 자연스러운 전환
- **절대 위치**: 썸네일 하단에 고정
- **반응형**: 화면 크기에 따라 자동 조정

### 표시 정보
1. **파일명**: 상단, 흰색, 중간 굵기
2. **카테고리**: 하단, 연한 회색, 폴더 아이콘 포함

---

**이 어두운 영역은 사용자가 썸네일만 보고도 파일 정보를 빠르게 파악할 수 있도록 도와주는 중요한 UI 요소입니다!**
