# Bundle 상품 등록 및 이벤트 배너 설정 가이드

## 📦 Bundle 상품 등록 방법

### 1단계: 관리자 페이지 접속
1. 관리자 계정으로 로그인
2. **Admin Dashboard** → **상품 관리** (또는 `/admin/products`) 클릭

### 2단계: Bundle 상품 등록
1. **"Add Product"** 버튼 클릭
2. **Category** 선택: **"📦 Bundle (묶음 상품)"** 선택
3. 기본 정보 입력:
   - **상품명**: 예) "특별 이벤트 Bundle"
   - **가격**: Bundle 전체 가격
   - **이미지**: Bundle 대표 이미지 업로드
   - **설명**: Bundle 상품 설명

### 3단계: 묶음 상품 구성
1. **"📦 묶음 상품 구성"** 섹션에서 포함할 상품 선택:
   - **🏷️ Stickers**: 포함할 스티커 상품 선택
   - **📮 Stamps**: 포함할 스템프 상품 선택
   - **📱 Phone Cases**: 포함할 폰케이스 상품 선택
2. 각 카테고리에서 원하는 상품을 클릭하여 선택 (다중 선택 가능)
3. 선택한 상품 개수 확인

### 4단계: 저장 및 상품 ID 확인
1. **"Save"** 버튼 클릭하여 저장
2. 저장 후 상품 목록에서 등록한 Bundle 상품 확인
3. **상품 ID 복사** (이벤트 배너 링크에 사용)

## 🎨 이벤트 배너 설정 방법

### 1단계: Content Management 접속
1. **Admin Dashboard** → **콘텐츠 관리** (또는 `/admin/content`) 클릭
2. 상단 탭에서 **"Hero"** 섹션 선택

### 2단계: 이벤트 배너 슬라이드 추가
1. **"Add Slide"** 버튼 클릭
2. 이미지/동영상 업로드
3. 제목/부제목 입력 (예: "특별 이벤트", "Bundle 상품 할인")

### 3단계: 이벤트 배너 설정
1. **"📦 이벤트 배너 설정"** 섹션으로 스크롤
2. **"이벤트 배너로 사용"** 체크박스 체크
3. **링크 URL** 입력:
   
   **옵션 1: Bundle 전체 목록 페이지**
   ```
   /bundle
   ```
   - 모든 Bundle 상품을 보여주는 목록 페이지로 이동
   
   **옵션 2: 특정 Bundle 상품 커스터마이징 페이지**
   ```
   /bundle/customize?product=상품ID
   ```
   - 특정 Bundle 상품의 커스터마이징 페이지로 직접 이동
   - `상품ID`는 "상품 관리" 페이지에서 Bundle 상품을 등록한 후 확인 가능

4. **이벤트 기간 설정** (선택사항):
   - **이벤트 시작일**: 배너가 표시되기 시작할 날짜
   - **이벤트 종료일**: 배너가 표시를 중단할 날짜
   - 설정하지 않으면 항상 표시됨

### 4단계: 저장 및 확인
1. **"Add"** 또는 **"Update"** 버튼 클릭
2. 홈페이지에서 이벤트 배너 확인

---

## 📝 전체 프로세스 요약

1. **Bundle 상품 등록** (`/admin/products`)
   - Category: Bundle 선택
   - 포함할 상품 선택
   - 저장 후 상품 ID 확인

2. **이벤트 배너 생성** (`/admin/content` → Hero 섹션)
   - 슬라이드 추가
   - 이벤트 배너 설정 체크
   - 링크 URL 입력 (`/bundle` 또는 `/bundle/customize?product=상품ID`)
   - 이벤트 기간 설정 (선택)

3. **고객 경험**
   - 홈페이지 Hero 슬라이드에서 이벤트 배너 확인
   - 배너 클릭 → Bundle 페이지로 이동
   - Bundle 상품 커스터마이징 및 구매

---

## 💡 팁

- **Bundle 상품 ID 확인 방법**: 
  - 상품 관리 페이지에서 Bundle 상품 목록 확인
  - 브라우저 개발자 도구로 상품 ID 확인 가능
  
- **링크 URL 예시**:
  - 전체 목록: `/bundle`
  - 특정 상품: `/bundle/customize?product=prod_1234567890`

- **이벤트 기간 설정**:
  - 시작일만 설정: 해당 날짜부터 계속 표시
  - 종료일만 설정: 해당 날짜까지 표시
  - 둘 다 설정: 지정된 기간에만 표시
