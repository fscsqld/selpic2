# SelPic2 - 커스터마이징 쇼핑몰

고객이 직접 커스터마이징할 수 있는 전문적인 온라인 쇼핑몰입니다.

## 주요 기능

### 🎨 커스터마이징 기능
- **색상 선택**: 다양한 색상 옵션으로 상품을 개인화
- **사이즈 선택**: 다양한 사이즈 옵션 제공
- **텍스트 인쇄**: 원하는 텍스트를 상품에 인쇄
- **실시간 미리보기**: 커스터마이징 결과를 실시간으로 확인

### 🛒 쇼핑 기능
- **상품 검색**: 상품명과 설명으로 빠른 검색
- **카테고리 필터**: 카테고리별 상품 필터링
- **장바구니**: 커스터마이징된 상품을 장바구니에 추가
- **주문 처리**: 간편한 결제 및 주문 처리

### 📱 반응형 디자인
- **모바일 최적화**: 모든 디바이스에서 최적의 경험
- **현대적 UI**: 깔끔하고 직관적인 사용자 인터페이스
- **애니메이션**: 부드러운 전환 효과와 인터랙션

## 기술 스택

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 🚀 빠른 실행 방법

### 방법 1: 자동 실행 스크립트 (권장)
프로젝트 폴더에서 다음 파일을 더블클릭하세요:

**Windows 사용자:**
- `start-homepage.bat` - 전체 과정을 자동화한 배치 파일 ⭐
- `start-homepage.ps1` - PowerShell 버전 (더 상세한 출력)
- `run.bat` - 간단한 실행 파일

**실행 과정:**
1. ✅ 기존 Node.js 프로세스 자동 종료
2. ✅ 캐시 자동 정리
3. ✅ 의존성 패키지 자동 확인
4. ✅ 개발 서버 자동 시작
5. ✅ 브라우저 자동 열기 ( http://localhost:3000)

### 방법 2: 수동 실행

#### 1. 의존성 설치
```bash
npm install
```

#### 2. 개발 서버 실행
```bash
npm run dev
```

#### 3. 브라우저에서 확인
```
http://localhost:3000
```

### 🔧 문제 해결

**서버가 실행되지 않는 경우:**
1. `start-homepage.bat` 실행 (자동 해결) 
2. 또는 수동으로:
   ```bash
   taskkill /F /IM node.exe
   npm install
   npm run dev
   ```

**포트가 이미 사용 중인 경우:**
- 자동 실행 스크립트가 기존 프로세스를 자동으로 종료합니다.

## 프로젝트 구조

```
selpic2/
├── app/                    # Next.js App Router
│   ├── cart/              # 장바구니 페이지
│   ├── products/          # 상품 목록 페이지
│   ├── globals.css        # 전역 스타일
│   ├── layout.tsx         # 루트 레이아웃
│   └── page.tsx           # 홈페이지
├── components/            # 재사용 가능한 컴포넌트
│   ├── Header.tsx         # 네비게이션 헤더
│   ├── ProductCard.tsx    # 상품 카드
│   └── CustomizationModal.tsx # 커스터마이징 모달
├── lib/                   # 유틸리티 및 설정
│   └── store.ts          # Zustand 상태 관리
├── public/               # 정적 파일
└── package.json          # 프로젝트 설정
```

## 주요 컴포넌트

### Header
- 반응형 네비게이션
- 장바구니 아이템 수 표시
- 모바일 메뉴

### ProductCard
- 상품 정보 표시
- 커스터마이징 및 장바구니 버튼
- 할인 표시

### CustomizationModal
- 상품 커스터마이징 옵션
- 실시간 가격 계산
- 수량 선택

## 상태 관리

Zustand를 사용하여 다음 상태들을 관리합니다:

- **상품 목록**: 모든 상품 정보
- **장바구니**: 선택된 상품과 커스터마이징 옵션
- **커스터마이징**: 사용자의 커스터마이징 선택
- **UI 상태**: 모달, 로딩 상태 등

## 커스터마이징 옵션

현재 지원하는 커스터마이징 옵션:

1. **색상 (Color)**: 다양한 색상 선택
2. **사이즈 (Size)**: 다양한 사이즈 옵션
3. **텍스트 (Text)**: 사용자 정의 텍스트 입력
4. **이미지 (Image)**: 사용자 이미지 업로드 (향후 구현 예정)
