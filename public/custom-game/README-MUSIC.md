# 게임 배경음악 추가 방법

게임에 자신만의 배경음악을 추가하는 방법입니다.

## 📁 파일 위치

음악 파일을 다음 위치에 저장하세요:

```
public/custom-game/music.mp3
```

전체 경로:
```
c:\Users\fscsq\Desktop\selpic2\public\custom-game\music.mp3
```

## ✅ 단계별 가이드

### 1. MP3 파일 준비
- 배경음악으로 사용할 MP3 파일을 준비하세요
- 파일 이름을 정확히 **`music.mp3`**로 변경하세요 (대소문자 구분)

### 2. 파일 복사
- `music.mp3` 파일을 `public/custom-game/` 폴더에 복사하세요

### 3. 게임 새로고침
- 브라우저에서 게임 페이지를 새로고침하세요 (F5 또는 Ctrl+R)
- 음악 버튼을 클릭하면 저장된 음악이 재생됩니다!

## 🎵 음악 파일 요구사항

- **형식**: MP3
- **파일명**: `music.mp3` (정확히 이 이름이어야 함)
- **위치**: `public/custom-game/` 폴더
- **크기**: 제한 없음 (하지만 너무 크면 로딩이 느려질 수 있음)

## 💡 참고사항

- **파일이 있으면**: 저장된 MP3 파일이 재생됩니다
- **파일이 없으면**: 자동으로 생성된 간단한 멜로디가 재생됩니다
- **파일 교체**: 다른 음악으로 교체하려면 파일을 덮어쓰고 페이지를 새로고침하세요

## 🔍 문제 해결

### 음악이 재생되지 않아요
1. 파일 이름이 정확히 `music.mp3`인지 확인
2. 파일이 `public/custom-game/` 폴더에 있는지 확인
3. 브라우저 콘솔(F12)에서 에러 메시지 확인
4. 페이지를 완전히 새로고침 (Ctrl+Shift+R)

### 파일을 추가했는데도 생성된 음악이 재생돼요
- 파일 이름과 위치를 다시 확인하세요
- 브라우저 캐시를 지우고 새로고침하세요 (Ctrl+Shift+R)
