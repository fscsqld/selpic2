SELPIC TETRIS

Collect characters and match blocks!

🎮 How to Play

1

Move Blocks

Use ← → arrow keys to move blocks left and right. Use ↓ to drop blocks faster. On phone or tablet, use the on-screen direction buttons below the score bar (Pause is included there). Blocks fall automatically at a speed that increases with each level.

2

Match Blocks

Match exactly 3 blocks of the same content horizontally or vertically to clear them. Each block cleared gives random points (5-50 points). Match 3 blocks = 1 explosion!

3

Block Types

English Blocks (Blue): Match blocks with your English name. Korean Blocks (Orange/Red): Match blocks with your Korean name - collect all characters! SELPIC Blocks (Purple, "SP"): Match 3 to explode entire row or column! Random Blocks (Yellow, symbols): Cannot be matched - they act as obstacles.

4

Complete Levels

Complete 10 explosions (3 blocks = 1 explosion) to finish each level. Complete all 5 levels to unlock exclusive promo codes and get up to 15% discount! Each level has faster block falling speed.

5

Game Over

If blocks stack to the top row (row 0), the game ends. Try to clear blocks before they reach the top! Plan your moves carefully to create chain reactions.

💡 Tips & Block Features

  • Random blocks (yellow symbols like ★, ◆, ●) cannot be matched - they act as obstacles
  • SELPIC blocks (purple "SP") are powerful - match 3 horizontally to clear the entire row, or vertically to clear the entire column!
  • Korean blocks can be collected - try to collect all characters from your Korean name
  • Mission Progress: Each explosion (3 blocks matched) = 10% progress. You need 10 explosions to complete a level
  • Scoring: Each block gives random points (5-50 points) when cleared
  • Pause: press SPACE or P, tap Pause in the sidebar, or use the touch Pause button below the score bar on mobile/tablet
  • Plan your moves to create row/column explosions with SELPIC blocks for maximum effect
